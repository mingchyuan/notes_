## 備料

- 蘋果放大冷藏，備水果杯只拿必要量。
- 優先使用廚房冰箱的冷凍鳳梨片。鳳梨罐頭開罐後，剩餘用六兩袋分裝，放至廚房冰箱。
- 紫長春鮮奶油放在大冷藏。1000g 鮮奶油配上 7g 鹽巴，最高 3 速打至 6 分發，拿起攪拌頭時呈彎鉤狀。

## 清洗咖啡機

### 午班收班

用清水洗牛奶管。

### 晚班收班

1. 拆除零件：長管、短管、三個奶管零件
2. 桶內裝 RO 水 1L 與藥水 50cc，並浸泡零件 15 分鐘。
3. 以清水刷洗零件。
4. 長按按鍵 0，接著按 1→5→1→3→1→1，選項為「咖啡與牛奶」。
5. 清空咖啡渣槽。
6. 加入藥錠後，按 0 進行第一階段清洗。
7. 將奶管放入藥水中，按 0 進行第二階段清洗。
8. 將藥水換成清水，按 0 進行第三階段清洗。
9. 按五次 0 回初始頁面。
10. 沖洗並晾乾咖啡渣槽。
11. 關機。
